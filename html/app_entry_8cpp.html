Descrição detalhada

Entry point for the application.

This file contains the main function which serves as the entry point for the application. It handles user commands to manage players and play the game.

The following commands are supported:

  • "FS": Finalize the system and exit the application.
  • "CA": Load players from a save file.
  • "CJ": Create a new player.
  • "RJ": Remove an existing player.
  • "LJ": List all players.
  • "EP": Play the game.

The commands are case-insensitive.

int Returns 0 upon successful execution.
